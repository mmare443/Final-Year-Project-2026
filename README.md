## Project Overview

This project aims to develop a Cloud-Based Student Management System hosted on Microsoft Azure.

The system will manage:

- Student records
- Course registration
- Attendance
- GPA calculations
- Results management
- Communication portal
- Accommodation management
- Reporting and analytics
